# SJDW Theme
A simple, clean and responsive WordPress template from the house of SJDWorld.

### Requirements
Following are the minimum requirements:

- [Wordpress](https://wordpress.org/) version: 5.6 or higher
- [PHP](https://php.org/) version: 7.3 or higher

### Installation
Install this theme into your Wordpress instance by:

1. Download and Extract the theme archive file.
1. Move the extracted theme folder into `wp-content/themes` directory.
2. Go to the `Appearance` menu in WordPress admin and activate the theme.

### Development
Following are the minimum requirements:

- [Composer](https://getcomposer.org/) version: 2.0 or higher
- [Node](https://nodejs.org) version: 16.0 or higher

Clone or fork the repository in your local and run the following command.

```sh
$ git clone <your-repo-url>
$ cd <your-repo-name>
$ composer install
$ npm install
```
